# Contributing to RunsOn Server

External contributions are welcome. RunsOn Server is commercial software, so a Contributor License Agreement (CLA) is required before changes can be merged.

## Signing the CLA

The current version is [CLA-v1.md](./contributors/CLA-v1.md).

When you open a pull request, the CLA Assistant workflow will ask you to sign by posting the following comment:

`I have read the Contributor License Agreement (CLA) for RunsOn Server and hereby sign it.`

Pull requests will not be merged until the CLA is signed.

If you are contributing on behalf of a sponsoring customer, the pull request must be submitted by a user authorized under that customer's sponsorship license, or otherwise separately approved by RunsOn.
